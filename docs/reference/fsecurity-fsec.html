xpathInject
Signature: Gen<string>

XPath input generator that provides malformed XPath inputs that can be used to discover XPath vulnerabilities. XPath is a "simple" language to locate information in an XML document. Similar to SQL Injection, XPath Injection attacks occur when an application uses user-supplied information to construct an XPath query for XML data. By sending intentionally malformed information into the application, an attacker can find out how the XML data is structured, or access data that he may not normally have access to. (more info: https://www.owasp.org/index.php/XPATH_Injection)

Severity: Critical
Vulnerability Indicators: incorrect/invalid syntax, syntax error, runtime errors, XPathException, System.Xml.XPath

CompiledName: XPathInject
